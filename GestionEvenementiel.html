Site web de gestion d'évenementiel

Ce projet a été conçu dans le cadre de la Compétition Québécoise de l'Ingénieurie(CQI).

Elle avait pour but de mettre en place une plateforme de gestion d'evenementiel. Les besoins du client étaient les suivantes:
1. L'organisateur devait-être en mesure assigné des tâches aux employés,
2. Modifier une tâche ou l'assigné à un autre employé,
3. Visualiser à temps réel l'évolution d'une tâche,
4. Sur l'interface, l'organisateur devrait voir les tâches par ordre de priorité,
5. En cas de perte d'objects, les invitées devaient avoir accès à un formulaire sur l'application, contenant les informations de l'objet.
6. Ils devaient également remplir un autre formulaire pour déclarer l'objet retrouvé.

Langages utilisées: Javascript, HTML, CSS, react
